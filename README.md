# Nano Banan Bulk Script 🍌

A Python script designed to process images in bulk using Google GenAI models (such as Gemini 3.1 Pro, Gemini 2.5 Pro, Flash, etc.).

The script takes all images from a source folder, applies a prompt (instruction) that you define, sends the query to the selected model, and saves the generated response in a `.txt` text file in your destination folder.

## Prerequisites

1. Download and install Python (version 3.9 or higher recommended).
2. Make sure you have a Gemini API Key from [Google AI Studio](https://aistudio.google.com/).
3. Open the `nano_banan_2.py` file with any text editor and paste your API Key in the configuration section at the top:
   ```python
   # Replace 'YOUR_API_KEY_HERE' with your actual Gemini API Key from Google AI Studio.
   API_KEY = "YOUR_API_KEY_HERE"
   ```

## Installation

Install the required dependencies included in the repository using `pip`:

```bash
pip install -r requirements.txt
```

*(The main dependencies are the official SDK `google-genai` and `pillow`).*

## Usage

The main script is named `nano_banan_2.py`. You can run it from the terminal by passing the required arguments:

```bash
python3 nano_banan_2.py --source /path/to/source --destination /path/to/destination --prompt "Your instruction here"
```

### Arguments and Options:

- `-s`, `--source` **(Required)**: The path to the folder containing the images to process. The script supports formats such as `.png`, `.jpg`, `.jpeg`, `.webp`, `.heic`, and `.heif`.
- `-d`, `--destination` **(Required)**: The path to the folder where the `.txt` results will be saved. If the folder does not exist, the script will create it automatically.
- `-p`, `--prompt` **(Required)**: The instruction that the AI model will execute on each processed image.
- `-m`, `--model` *(Optional)*: The AI model you want to use. Defaults to `gemini-3.1-pro`.

### Examples

**1. Basic Usage:**
```bash
python3 nano_banan_2.py \
  --source ./input_images \
  --destination ./results \
  --prompt "Describe in detail all the objects you see in this image"
```

**2. Advanced Usage (Selecting a faster or lower-latency model):**
```bash
python3 nano_banan_2.py \
  --source ./my_photos \
  --destination ./analysis \
  --prompt "Extract all the text from the image" \
  --model gemini-3.1-flash
```

## Output and Results

If the source folder contains an image named `landscape_photo.jpg`, after successful execution, you will find a file named `landscape_photo.txt` in your destination folder that exclusively contains the response and analysis generated by the model.
